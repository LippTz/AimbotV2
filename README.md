## 🚀 Overview
KreinAim adalah script Roblox ESP + Aimbot lengkap dengan performa stabil dan tampilan clean.

---

## ✨ Features

### 👁️ ESP
- Skeleton ESP
- Box ESP
- Tracer ESP
- Name + Distance + HP
- Health Bar
- Team Check

### 🎯 Aimbot
- Auto Lock Target
- Smooth Aim
- Multi Lock Part
- FOV System
- Wall Check
- Alive Check

### 🔵 FOV
- Circle FOV
- Dual Render (Drawing + GUI)
- Adjustable Size & Thickness

### 🌈 Visual
- RGB Effect
- Clean UI (Rayfield)

---

## 📱 Compatibility
- PC
- Mobile
- All Executors (Drawing / Non-Drawing)
